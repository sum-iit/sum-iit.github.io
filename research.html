Research Interests

My research focuses on the intersection of software engineering and distributed systems. I am particularly interested in:

  • Scalable web architecture patterns and microservices design
  • Performance optimization in cloud-native applications
  • Machine learning integration in production systems
  • Real-time data processing and stream computing

Publications

Optimizing Microservices Performance in Cloud Environments

Sumit Sah, Jane Smith, Robert Johnson

Proceedings of the International Conference on Software Engineering, 2024

This paper presents novel optimization techniques for microservices architectures deployed in cloud environments, achieving up to 40% improvement in response times.

Real-time Data Processing at Scale: Patterns and Practices

Sumit Sah, Michael Chen

Journal of Distributed Computing, Vol. 15, 2023

An extensive study of real-time data processing patterns, with practical implementations and performance benchmarks across different streaming platforms.

Machine Learning Model Deployment in Production: A Systematic Approach

Sumit Sah, Dr. Sarah Williams, Alex Kumar

Conference on Machine Learning Systems, 2023

A comprehensive framework for deploying and monitoring machine learning models in production environments with focus on reliability and scalability.
